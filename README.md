## Ejecución

En este repositorio se encuentra implementado el servidor GraphQL, para inicializarlo ejecutar los siguientes comandos

<pre>
<code>
cd server
nvm use (linux)
nvm use 19.0.0 (windows)
npm i
npm start
</code></pre>
